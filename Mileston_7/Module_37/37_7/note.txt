üîπ Definition:

localStorage is a web storage object in JavaScript that allows you to store key-value pairs in the browser.

Data persists even after the browser is closed.

Values must be strings, so objects need to be converted using JSON.stringify().

üîπ Example Breakdown:

    localStorage.setItem(id, JSON.stringify(data))
}


id and name are values from the input fields.

data is an object: {id: '101', name: 'Maruf'}

localStorage.setItem(id, JSON.stringify(data))

Saves the object as a string under the key = id.

console.log(data) prints the object before storing.

2Ô∏è‚É£ Retrieving data from localStorage
const storedItem = localStorage.getItem('101')
console.log(JSON.parse(storedItem))


localStorage.getItem('101') fetches the string stored with key '101'.

JSON.parse() converts the string back to an object.

Output example: {id: "101", name: "Maruf"}

Important: This works only if key '101' exists. Otherwise, it returns null.

3Ô∏è‚É£ Clearing localStorage
